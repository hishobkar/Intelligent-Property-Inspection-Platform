ℹ Architecture

Event-Driven Microservices — Five independent services communicate asynchronously.

  • Property Service (ASP.NET Core 9, CQRS+MediatR) — CRUD for property records
  • Inspection Service (ASP.NET Core 9 Minimal API) — Schedules and completes inspections
  • Risk Scoring Service (Python FastAPI) — Multi-factor weighted risk engine
  • Document Service (ASP.NET Core 9) — Generates compliance and inspection reports
  • Notification Service (Python FastAPI) — Multi-channel event-driven notifications

All services expose OpenAPI/Swagger documentation and health endpoints. SQLite is used for this demo; SQL Server / CosmosDB in production.
